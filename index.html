Git

  • To see which branch we're in use the command "git status"
  • To create a new feature branch use there command "git checkout -b NEW NAME" Git checkout tells it to switch branches, -b tells it a new branch followed by the new branch name
  • To record changes to a file use the command "git add -A"
  • To commit changes use the command "git commit -m 'Detailed message about change'"
  • Pull to ensure latest commits/changes use the command "git pull origin main" Pulling is to put a change into a local setting. Origin is telling it that its in GitHub Reopsitory and main is tellig it the branch.
  • To push out changes use the command "git push origin 'Branch name'" Pushing takes it from local into GitHub's cloud. This is also another way to create a new branch.
  • Using a - followed by a letter is what is called a flag in Git
